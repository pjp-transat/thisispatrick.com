Structured Intelligence for Sales Commissions

Preventing Leaks in OpEx while Saving Hours of Analyst Time

A Python workflow which identifies discrepancies between expected and actual sales payouts and recommends a course of action for each mismatch. Developed using Python, SQLite, and Gemini 2.0 Flash-Lite.

The Operational Win: Implemented both a Gen-AI (primary) and rules-based (secondary) interpretation layer for all quarterly deals, eliminating the need for manual deal reviews while handling API instability gracefully.

The Logic Engine (Python)

The core script used to process the commission logic and interface with the LLM.

The Output (Board-Ready Audit)

The resulting artifact automatically flags variances and drafts both a CFO-summary and a rep-facing slack message.
